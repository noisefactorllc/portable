# gradientSweep

A diagonal color gradient that rotates over time.

## Parameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| speed | float | 1.0 | 0-4 | Rotation speed multiplier |

## Usage

```
search user

gradientSweep(speed: 1.0)
  .write(o0)

render(o0)
```
